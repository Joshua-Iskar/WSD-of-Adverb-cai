## 各标签的含义

| 标签 | 含义         |
|------|--------------|
| 0    | 表示才华，才能 |
| 1    | 表示减值，即“才-” |
| 2    | 表示某事发生得晚 |
| 3    | 表示某种逻辑关系 |
| 4    | 强调事实 |

## 判断依据

| 标签 | 描述                                |
|------|-------------------------------------|
| 0    | 略                                  |
| 1    | 刚刚才 <br> 才+数量 <br> 才……就    |
| 2    | 直到……才                            |
| 3    | 条件：只有……才、除非……才 <br> 目的：为了……才 <br> 原因：因为……才、正是……才、经过……才 |
| 4    | 略                                  |
